# stellar-contract-verification

Contract source verification service for Stellar — built with Next.js (frontend) and Rust/Axum (backend).

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

The frontend requires the Rust backend running on port `8088`. See `backend/SETUP.md` for backend setup instructions.
